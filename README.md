# Project List

A node graph pathfinding project about efficient navigation through interconnected nodes under
a set of strict constraints and optimization goals.

## A_maze_ing (Python)
A procedural maze generation and solving project that creates randomized mazes and applies pathfinding algorithms to find optimal solutions.


## Push_swap (C)
A sorting algorithm optimization project that implements efficient sorting methods using limited stack operations.
